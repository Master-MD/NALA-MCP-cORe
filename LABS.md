# Labs

Labs are experimental.

Main rule:

```text
Labs do not directly modify the live vault.
```

The app can create a sandbox check that copies:

- `nala.snapshot.sqlite`
- `events.snapshot.jsonl`
- `input/`
- `output/`
- `report.json`
- `report.md`
- `checksums.sha256`
- `logs/`
